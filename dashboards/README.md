Dashboards

This folder contains dashboard assets created to communicate business insights
from the car sales data.

The primary dashboard was built using Google Looker Studio and focuses on:
- Sales performance KPIs
- Profitability by vehicle make and condition
- Mileage and pricing trends

A public dashboard link is provided in the main project README.

Dashboard Export

A PDF export of the Looker Studio dashboard is included for quick review
without requiring access to Looker Studio.
